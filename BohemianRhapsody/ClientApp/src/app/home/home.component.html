Bohemian Rhapsody

Architecture

Api - The api is written in C# using asp.net.mvc.

- The main database layer is the SqlLiteData object that implements the IMusicData interface

- The Api Controllers are split up according to models Genre, Song, Artist and Album

- The Api uses Rest and JSON formatting

FrontEnd - The front end is written in Angular with bootstrap and sass

RequireMents

You can use any frontend tech that you’re comfortable with (React, Angular etc)

Backend API – c# mvc

Data must be stored in any database of your choosing - Local SQLLite database

Sass or Less - Sass, please checkout the sass buttons below.

Redux or RXjs - No Redux, just a lot of subscriptions, maps and pipes

Bootstrap, Foundation or Material - Bootstrap stock

Responsive Design - Mobile first

SOLID Principles - You bet

Dependency Injection - Implemented IMusicData Interface injection into SQLLiteData

Unit tests with Mock - Please checkout the tests paragraph

BONUS: Docker - Sorry No Docker, yet ..

Assessment code in Github or Gitlab - My Github link is https://github.com/graham136/ASP.Net.Angular
